Overview

We propose d1, a two-stage post-training framework for masked diffusion LLMs (dLLMs), combining supervised finetuning (SFT) and a novel reinforcement learning algorithm diffu-GRPO. Our work adapts GRPO to dLLMs using a novel log-probability estimator and random prompt masking, significantly improving reasoning capabilities.

Main Results

d1-LLaDA significantly improves reasoning performance over the base model and other training recipes.

Training Framework

d1 first applies masked SFT using a high-quality reasoning dataset. Then, we apply diffu-GRPO, a policy gradient method tailored for masked dLLMs, enabling efficient log-probability estimation through a single forward pass and stochastic prompt masking.

Stage 1: Masked SFT on s1k dataset.

Efficient log-probability estimation via random prompt masking.

Stage 2: diffu-GRPO enables scalable and efficient RL training for dLLMs.

Random Masking as Regularization

Random masking improves efficiency and reduces the number of required online generations.

State-of-the-Art Comparison

d1-LLaDA achieves top GSM8K and competitive MATH500 scores compared to leading models.

Qualitative Reasoning Traces

Models trained with d1 show self-correction behaviors (“aha moments”).
